# Employee Wellness Management

## Project Overview
This project is a secure Employee Wellness Management System developed using Streamlit and PostgreSQL.It provides secure user authentication with email verification and password reset using OTP.

## Features
- User Registration
- User Login
- Email Verification (OTP)
- Forgot Password
- Password Reset
- JWT Authentication
- PostgreSQL (Neon Database)

## Technologies Used
- Python
- Streamlit
- PostgreSQL (Neon)
- JWT
- Gmail SMTP
- GitHub
